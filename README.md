# @paragraphcms/parser-core

Shared parser engine for Paragraph CMS.

This package is the shared layer used by framework-specific Paragraph CMS renderers.

## Exports

- shared content and page types
- default class name, markdown, and sanitization settings
- content input resolution helpers
- content format inference helpers
- `markdown | html | tiptap` to sanitized HTML rendering
- shared root and slot class name resolution
